# 故障树智能生成系统 - 使用说明

## 环境要求
- Conda环境：fsy（Python 3.9+）
- Node.js：16+
- 可选：PostgreSQL 12+、Neo4j 4.0+

## 快速开始

### 第一步：安装依赖
双击运行 `安装依赖.bat`，会自动：
1. 激活conda环境fsy
2. 安装后端Python依赖
3. 安装前端Node.js依赖

### 第二步：启动系统
双击运行 `快速启动.bat`，会自动打开两个窗口：
- 窗口1：后端服务（http://localhost:8000）
- 窗口2：前端服务（http://localhost:3000）

### 第三步：使用系统
1. 打开浏览器访问：http://localhost:3000
2. 上传工业设备文档（DOCX/Excel/CSV/TXT）
3. 提取故障知识
4. 生成故障树
5. 编辑和导出

## 手动启动（开发模式）

### 启动后端
```bash
conda activate fsy
cd backend
python main.py
```

### 启动前端
```bash
cd frontend
npm run dev
```

## 常见问题

### Q1: conda环境fsy不存在？
```bash
conda create -n fsy python=3.9
conda activate fsy
```

### Q2: 端口被占用？
修改配置：
- 后端：backend/main.py 中的 port=8000
- 前端：frontend/vite.config.js 中的 port: 3000

### Q3: 数据库连接失败？
编辑 backend/.env 文件，配置数据库连接信息
如果不需要数据库，系统会使用模拟数据运行
